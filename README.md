# Hedera Blockchain App

A blockchain application built on the Hedera network.

## Technologies used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting started
```sh
# Clone the repository
git clone https://github.com/Hodlsensei/hedera-blockchain-app.

# Navigate to the project directory
cd my-hedera-app

# Install dependencies
npm install

# Start the development server
npm run dev
```
